# Substrates JMH Benchmark Comparison

## Humainary (Alpha) SPI vs Fullerstack Implementation

This document compares JMH benchmark results between:
- **Humainary Alpha SPI**: `io.humainary.substrates.spi.alpha.Provider` (reference implementation)
- **Fullerstack SPI**: `io.fullerstack.substrates.CortexRuntimeProvider` (production implementation)

## Test Environments

| Property | Humainary | Fullerstack |
|----------|-----------|-------------|
| **Platform** | Mac mini (M4) | GitHub Codespaces (Linux) |
| **Chip/CPU** | Apple M4 (10 cores) | Intel Xeon (cloud) |
| **Memory** | 16 GB | Variable (cloud) |
| **Java Version** | 25.0.1 LTS | 25.0.1-open |
| **JMH Version** | 1.37 | 1.37 |
| **Warmup** | 3 iterations, 1s each | 3 iterations, 1s each |
| **Measurement** | 5 iterations, 1s each | 5 iterations, 1s each |

**Note**: Direct comparison is affected by different hardware. Humainary's M4 is significantly faster than cloud-based Intel Xeon. Relative patterns within each implementation are more meaningful than absolute numbers.
